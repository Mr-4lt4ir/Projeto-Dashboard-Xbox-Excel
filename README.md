# Projeto-Dashboard-Xbox-Excel
Repositório do desafio Criando um Dashboard de Vendas do Xbox com Excel da DIO

# Sobre o projeto:
Esta dashboard tem como objetivo fornecer ao gestor uma visão das assinaturas do Xbox Game Pass ao longo do tempo, com análises em períodos mensal, trimestral e anual.

# Dados utilizados:
Para a base de dados, foi utilizado uma tabela contendo informações a respeito das assinaturas do Xbox Game Pass, como o tipo de assinatura, o valor total e o valor do cupom de desconto, se existente.

# Como foi produzido:
O gráfico da planilha Dashboard foi retirada da tabela criada na planilha Cálculos, onde esta tabela servia para mostrar o faturamento do total de vendas de planos anuais.

Já o Big number que mostra o total de subscriptions EA play season pass, foi retirada da tabela que mostra o Total assinaturas da EA play season pass.

Por fim, O Big number que mostra o total de subscriptions Minecraft season pass, foi retirada da tabela que mostra o Total assinaturas do Minecraft season pass.
